When starting a new piece of work, ask if the work should be done in a new git worktree. To facilitate working with worktrees, use the `git-wt` tool. To learn what commands to run, read `GIT_RT.md`. If work was done on a worktree and the work is now complete, remove the worktree to keep the local repo clean.

When opening a new PR in GitHub, read `NEW_PR.md`.
